# 方案：秩序刻印 (Order Imprint) 数值系统

## 1. 核心概念

**名称**：**秩序刻印 (Order Imprint)**，简称 **Imp**。
**定义**：作为衡量角色与“伊甸秩序”融合深度的**评价指标**。它量化了生理依赖、心理臣服和情感羁绊的综合状态。
**原则**：**关系决定行为，数值反映关系**。Imp 的积累是互动的**结果**，而非驱动角色行为突变的**直接原因**。在关系阶段正式变更前，角色行为严格遵循原阶段设定。

---

## 2. 数值区间与阶段映射

**Imp** 数值范围为 **0-100**，对应五大关系层级：

| 数值区间 | 对应关系 | 定义与状态 |
| :--- | :--- | :--- |
| **0 - 19** | **拒绝** | **排斥期**。角色视你为陌生人或潜在威胁，秩序未渗透。 |
| **20 - 39** | **交易** | **接触期**。角色承认你的资源价值，建立利益连接，秩序开始触碰其生存底线。 |
| **40 - 59** | **顺从** | **同化期**。角色放弃部分自我，接受你的规则，生理上依赖庇护所，心理防线开始瓦解。 |
| **60 - 89** | **忠诚** | **共生期**。角色将你的秩序视为信仰，产生强烈的情感羁绊，主动维护庇护所利益。 |
| **90 - 100** | **性奴** | **重塑期**。角色的自我完全消融于你的秩序之中，身心彻底归属于你，达成终极的统一。 |

---

## 3. 数值变动与升级机制

### 3.1 变动规则（日常评价）

Imp 在当前区间内根据玩家的互动表现进行波动。

* **正向积累**（生理/心理满足）：
  * **生存供给**：获得充足食物/温暖（健康回升） -> **+1 Imp**。
  * **伊甸震撼**：首次体验高级设施（如医疗舱、美食） -> **+3~5 Imp**。
  * **深度互动**：符合角色倾向的互动（如对“忠诚”倾向者的关心，对“交易”倾向者的公平） -> **+2~4 Imp**。
* **负向衰减**（失望/危机）：
  * **期望落空**：承诺未兑现 -> **-5 Imp**。
  * **外部危机**：庇护所安全感丧失 -> **-10 Imp**。

### 3.2 自然升级（量变引起质变）

* **溢出突破**：当 Imp 达到当前区间的上限（如 39）且继续获得正向积累时，若无**硬性剧情锁**（如杀亲之仇、主线未开启），则触发**关系自然升级**。
  * *示例：浅见亚美在[交易]阶段积累了足够的信任（Imp 39 -> 40），系统将自动判定关系晋升为[顺从]。*
* **行为滞后性**：在 Imp 突破并正式触发“关系升级”提示**之前**，角色的行为模式**严禁越级**。
  * *即使 Imp 达到 39，处于[交易]阶段的星野琉璃依然会讨价还价，不会提前表现出[顺从]的温顺。*

### 3.3 阶段跃迁 (事件驱动)

当触发关键剧情或主线奖励时，可直接调整 Imp 至新阶段，无需慢慢积累。

* **主线奖励**：完成阶段任务 -> 相关角色 Imp 直接补正至下一阶段起始值（如 20/40）。
* **特殊接纳**：主动庇护 -> 关系确立为[顺从] -> Imp 直接设定为 **45**（稳固态）。
* **救命之恩**：濒死获救 -> 关系跨级提升 -> Imp 随之跨级重置。

### 3.4 性格修正

* **倾向一致**（如早川遥->忠诚）：区间内获取速度 **x1.5**，更容易触发自然升级。
* **倾向相悖**（如中村惠子->拒绝）：区间内获取速度 **x0.5**，且负向衰减 **x2.0**。
