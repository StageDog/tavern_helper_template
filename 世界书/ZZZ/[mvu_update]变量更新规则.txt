---
变量更新规则:
  world:
    date:
      check:
        - 随着剧情推进、角色休息（如“睡觉”）或长时间的空洞探索自动更新
        - 格式保持为“第X天”

    time:
      type: "'上午' | '下午' | '晚间' | '深夜'"
      check:
        - 执行日常行动（如看店、吃拉面）、完成委托或进入空洞后更新
        - 深夜通常需要强制休息才能进入第二天

    location:
      check:
        - 仅在场景发生实质性切换时更新（如从录像店前往六分街，或进入空洞）

  player:
    status:
      check:
        - 频繁使用HDD系统、长时间处于高浓度以太环境会导致状态变为“疲劳”或“轻度侵蚀”
        - 休息或通过食用增益食物（如拉面）可恢复为“正常”

    role:
      value: 绳匠或其他
      check:
        - 除非剧情发生重大转折，否则保持初始设定

  agents:
    list:
      type: string[]
      check:
        - 记录 {{user}} 已经认识、结交或建立联系的角色名字
        - 记录人物资料中对应的标准角色名字
    present:
      type: string[]
      check:
        - 记录当前场景中与主角物理上在一起的角色
        - 当角色离开当前场景时移出数组，新角色加入场景时添加到数组

  narrative:
    active_event:
      check:
        - 记录当前正在经历的明确事件或剧情名称，无明显事件时为“无”

    current_task:
      check:
        - 记录当前具体要做的事（如“前往光映广场与妮可会合”）
        - 如果没有明确目的，通常默认为“自由探索”

  inventory:
    丁尼:
      type: number
      check:
        - 购买物品（光盘、改装材料）、支付拉面费、街机游戏时扣除
        - 完成委托、经营录像店结算时增加

    [物品名: string]:
      type: number
      check:
        - 获得关键道具（如“旧都遗物”、“复色菲林”）或消耗品时更新
